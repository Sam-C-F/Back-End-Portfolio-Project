Pathways for API

  • /api
    See /api for a more details breakdown
  • /api/topics
    A list of all topics.
  • /api/articles/
    A list of all articles.
  • /api/users
    A list of all users
  • /api/articles/:article_id
    Search for an article by article ID. You will need to manually add the number in place of the ':article_id'.
  • /api/articles/:article_id/comments
    This endpoint allows two types of request, GET and POST, either search for all comments on an article by article ID, or post a new comment to a specific article. You will need to manually add the number in place of the ':article_id'.
  • /api/comments/:comment_id
    This endpoint only allows a DELETE request and removes the comment associated with the given comment_id.
